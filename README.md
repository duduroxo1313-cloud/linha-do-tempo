 📖 Descrição do Projeto

Este projeto apresenta uma linha do tempo dos conteúdos estudados na disciplina de Desenvolvimento Web durante o curso de Desenvolvimento de Sistemas do SENAI. O objetivo é demonstrar a evolução do aprendizado, desde os conceitos iniciais de HTML e CSS até a construção de um Projeto Integrador.

 🗓️ Linha do Tempo das Aulas

 Aula 01 - Introdução ao Desenvolvimento Web

* Conceitos básicos da Web.
* Estrutura de sites e funcionamento da internet.

 Aula 02 - HTML Básico

* Estrutura de documentos HTML.
* Principais tags e elementos.

 Aula 03 - Formulários HTML

* Criação de formulários.
* Campos de entrada e botões.

 Aula 04 - CSS Básico

* Introdução ao CSS.
* Estilização de páginas.

 Aula 05 - Box Model

* Margens, bordas, preenchimentos e dimensões dos elementos.

 Aula 06 - Flexbox

* Organização e alinhamento de elementos em layouts flexíveis.

 Aula 07 - Grid Layout

* Criação de layouts em grade.
* Distribuição de elementos em linhas e colunas.

 Aula 08 - Responsividade

* Adaptação das páginas para diferentes dispositivos.

 Aula 09 - Media Queries

* Aplicação de estilos específicos para diferentes tamanhos de tela.

 Aula 10 - Animações CSS

* Criação de efeitos visuais utilizando CSS.

 Aula 11 - Keyframes

* Desenvolvimento de animações avançadas com `@keyframes`.

 Aula 12 - Projeto Integrador

* Aplicação dos conhecimentos adquiridos na construção de um projeto completo.
