Timeline do Semestre

Aula 01

Introdução ao Desenvolvimento Web.


O desenvolvimento web é a área da programação voltada para a criação de sites e aplicações para a internet, utilizando tecnologias como HTML, CSS e JavaScript.

Aula 02

HTML Básico e estrutura de páginas.


O HTML é a linguagem utilizada para estruturar páginas da web, organizando elementos como títulos, parágrafos, imagens e links.

Aula 03

Formulários HTML.


Os formulários HTML permitem coletar informações dos usuários por meio de campos como texto, e-mail, senha e botões de envio.

Aula 04

CSS Básico e estilização.


O CSS é utilizado para definir a aparência das páginas web, permitindo personalizar cores, fontes, tamanhos e o posicionamento dos elementos.

Aula 05

Box Model e espaçamentos.


O Box Model define a estrutura dos elementos em uma página, incluindo conteúdo, margens, bordas e espaçamentos internos, permitindo organizar melhor o layout.

Aula 06

Flexbox para layouts modernos.


O Flexbox é um recurso do CSS que facilita a organização e o alinhamento dos elementos, permitindo criar layouts modernos e responsivos.

Aula 07

Grid Layout.


O Grid Layout é um recurso do CSS que permite organizar elementos em linhas e colunas, facilitando a criação de layouts estruturados e responsivos.

Aula 08

Responsividade.


A responsividade permite que as páginas web se adaptem a diferentes tamanhos de tela, proporcionando uma melhor experiência em computadores, tablets e celulares.

Aula 09

Media Queries.


As Media Queries são regras do CSS que permitem aplicar estilos diferentes com base nas características do dispositivo, como tamanho da tela e orientação.

Aula 10

Animações CSS.


As animações CSS permitem criar transições suaves e efeitos visuais, melhorando a experiência do usuário e tornando as páginas mais dinâmicas.

Aula 11

Keyframes e efeitos visuais.


Os keyframes são utilizados para definir etapas específicas de uma animação, permitindo criar efeitos visuais complexos e personalizados.

Aula 12

Projeto Integrador.


O projeto integrador é uma oportunidade para aplicar os conhecimentos adquiridos ao longo do semestre, desenvolvendo um site completo e funcional.

Minha Evolução



Quem eu era no início do semestre

No começo do semestre eu tinha apenas uma noção básica sobre tecnologia e internet. Meu conhecimento sobre desenvolvimento web era bastante limitado e eu não compreendia como os sites eram construídos. Sabia apenas navegar em páginas da internet e utilizar aplicativos do dia a dia. Minhas expectativas eram aprender a criar páginas simples, entender como funcionava a estrutura de um site e desenvolver habilidades que pudessem ser úteis para minha formação acadêmica e profissional.

Quem eu sou agora

Após vários meses de estudos, adquiri conhecimentos importantes sobre HTML5 e CSS3. Aprendi a criar a estrutura semântica de páginas utilizando tags como header, nav, main, section, article e footer. Também desenvolvi habilidades relacionadas à criação de formulários, organização de conteúdo, utilização do Box Model, aplicação de cores, fontes e estilos personalizados.

Além disso, aprendi a utilizar Flexbox e Grid Layout para construir interfaces mais modernas e organizadas. Compreendi a importância da responsividade e das Media Queries para adaptar os projetos a diferentes tamanhos de tela. Outro aprendizado importante foi o uso de animações com CSS e keyframes.

Meu próximo objetivo

Meu próximo objetivo é aprofundar meus conhecimentos em desenvolvimento front-end, aprendendo JavaScript para criar páginas dinâmicas e interativas. Também pretendo estudar frameworks modernos, melhorar minhas habilidades de design de interfaces e desenvolver projetos cada vez mais completos.
